# Code of Conduct

## Our Pledge

我们致力于为所有贡献者提供一个友好、尊重、包容的协作环境。

## Expected Behavior

- 尊重不同背景与观点。
- 针对问题讨论，不进行人身攻击。
- 接受建设性反馈并保持专业沟通。

## Unacceptable Behavior

- 骚扰、歧视、侮辱或威胁性言论。
- 泄露他人隐私信息。
- 持续发布与项目无关、破坏协作的内容。

## Enforcement

维护者有权对违反本准则的行为采取警告、限制或移除权限等措施。

## Contact

如遇到行为问题，请通过仓库维护者提供的私下渠道联系处理。
